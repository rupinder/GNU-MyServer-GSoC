WinCGI

WinCGI is a method for writing CGI scripts in Visual Basic. It's not a "web standard", as it will work only on servers running in Windows environment.

To enable WinCGI in MyServer, you need to configure the correct MIME Type corresponding to your WinCGI script files, and make them run as WinCGI (EXECUTEWINCGI)

After configuring your MIME Typer properly, all you need to do to run your WinCGI script, is to use the script as the 'action' of your web form, like in this example:

<FORM METHOD=POST ACTION=www.myfiles.com/cgi-bin/mywincgi.exe>

With this, MyServer will decode and send the data of your form to the WinCGI script.
