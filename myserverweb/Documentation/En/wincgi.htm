WINCGI


WinCGI allows to have communication with the new process trough .ini files. WinCGI is supported only on the Microsoft Windows platform.
To enable WinCGI in MyServer, you need to configure the correct MIME type corresponding to your WinCGI application files, and make them run as WinCGI (EXECUTEWINCGI)

Like in CGI, you can place all your applications in a cgi-bin directory in the web directory in MyServer as seen below:


The cgi-bin folder is directly beneath the selected web folder.



However, WinCGI applications can be placed anywhere in the web directory.
