HTTP

HTTP (Hypertext Transfer Protocol) is the set of rules that govern thetransferring of files over the Internet. These files can be text,graphics, sound or video.  HTTP uses a simple server-clientinterface, where the client requests a file or application and theserver processes the request and returns the correct file orapplication.

he HTTP protocol is the de facto standard for transferring documents over the World Wide Web.

The current version of HTTP is HTTP 1.1.  More information can befound at the WWWConsortium.
