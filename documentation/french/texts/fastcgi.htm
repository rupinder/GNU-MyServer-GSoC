FASTCGI


FastCGI est une extension de CGI, indépendante du langage utilisé, offrant des grandes performances pour les programmes, en n'utilisant pas les APIs spécifiques du serveur. Avec FastCGI, vous n'aurez pas un processus séparé pour chaque requête de script. La communication avec le processus MyServer se fait par des sockets.

Vous pouvez utiliser FastCGI dans MyServer, simplement en spécifiant le Type MIME de vos langages de script pour fonctionner comme FastCGI (mis à RUNFASTCGI).

Lisez la section PHP pour savoir comment utiliser PHP+FastCGI.

Serveurs FastCGI Distants

MyServer peut utiliser un serveur FastCGI distant. Le type MIME suivant utilise une connecxion distante:
<MIMETYPE>
<EXT>php</EXT>
<MIME>text/html</MIME>
<CMD>RUNFASTCGI</CMD>

<MANAGER>@host:port</MANAGER>
</MIMETYPE>

La définition d'un gestionnaire distant doit commencer par @.
Si vous configurez les types MIME par le biais du Panneau de contrôle, vous pouvez ajouter un serveur distant en ajoutant une ligne de la forme
@HOST:PORT pour le chemin de FastCGI.


Référez vous au site officiel de FastCGI pour plus d'informations sur FastCGI.
