Subversion


MyServer est constamment en développement. Ainsi, si vous voulez avoir la version la plus récente, vous devrez utiliser le système Subversion. Les versions disponibles sur le serveur offrent souvent plus de fonctionnalités que celles disponibles en téléchargement. Cependant, les versions proposées au public ont été soumises à des tests supplémentaires pour s'assurer de leur stabilité. Nous vous recommandons donc l'utilisation des sorties officielles pour un environnement professionnel.

Téléchargement d'un client Subversion

Si vous ne possédez pas de client Subversion, visitez le site http://subversion.tigris.org/.

Accès anonyme

Pour obtenir les dernières sources disponibles, utilisez le client en tapant une des lignes suivantes:


1.  La première fois que vous copiez le contenu du serveur svn, tapez:

svn co https://svn.sourceforge.net/svnroot/myserverweb/trunk/myserverweb myserverweb

2. Si vous voulez mettre à jour vos fichiers, tapez depuis votre répertoire web de MyServer:

svn update

Il existe des clients Subversion pour le système d'exploitation Windows. Un de ceux ci est le client Tortoise SVN.
