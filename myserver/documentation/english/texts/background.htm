MyServer Project

www.myserverproject.net



MyServer is a powerful web server designed, to be easily run on a personal computer by the average computer user.
It is based on multi-threaded architecture, which makes it extremely scalable and usable in large scale sites, as well as in small networks. It has built-in supports for the HTTP, HTTPS and FTP protocols.
Support for dynamic pages is made available trough CGI, ISAPI, WinCGI, SCGI and FastCGI protocols.
You can share files in minutes.

MyServer is available GNU/Linux and other Unix-like operating systems, as well as Windows.

Our principal goal is to create a free and powerful server, to allow anyone to transform a Personal Computer into a web server and share web files easily.
MyServer can also be used professionally due its event driven multi-threaded architecture and advanced features.

Please visit our site www.myserverproject.net for more information and the latest news about this project. You can use the site forum to post any questions or comments, you may have about the project.
