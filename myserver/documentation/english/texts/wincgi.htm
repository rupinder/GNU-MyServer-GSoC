WINCGI


WinCGI permet la communication avec le nouveau processus via des fichiers .ini. WinCGI n'est supporté que par la plate forme Windows.
Pour utiliser WinCGI dans MyServer, vous devez configurer correctement le type MIME correspondant à vos fichiers d'application WinCGI, et les lancer comme tel (EXECUTEWINCGI)
