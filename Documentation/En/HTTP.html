HTTP

HTTP (Hypertext Transfer Protocol) is the set of rules that govern the transferring of files over the Internet. These files can be text, graphics, sound or video.  HTTP uses a simple server-client interface, where the client requests a file or application and the server processes the request and returns the correct file or application.

Most of the files accessed are written in HTML language and may contain CGI or ISAPI applications.

The current version of HTTP is HTTP 1.1.  More information can be found at the WWW Consortium.
