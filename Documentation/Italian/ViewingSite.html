ACCEDI AL TUO SITO



Per vedere i tuoi files in locale digita:

http://localhost

nel tuo browser.
Per vedere i files attraverso internet, digita l'indirizzo IP del tuo computer. Per esempio se il tuo IP è 36.65.34.32 il tuo sito sarà visibile su:

http://36.65.34.32.

Nome DNS

Se ti piacerebbe avere un sito del tipo: http://william.homeip.net invece del tuo IP devi avere un nome DNS.


Un nome DNS è un meccanismo per assegnare ad un IP un nome mnemonico come www.google.com. Tu puoi usare un servizio gratuito come DynDNS.org. Questo servizio è per un PC con una connessione sempre attiva(ADSL, cable,...) accessibile per la maggior parte della giornata


Se non hai un IP statico ma cambia ogni volta che ti connetti puoi usare il servizio offerto da DynDNS.org.


Per essere sicuro ogni volta che si accede a internet di aggiornare il nome DNS si può usere un client DNS: http://www.directupdate.net.


Se non ti piace il nome: http://myName.homeip.net ma vorresti qualcosa come: http://www.myName.net allora si necessita della registrazione di un nome di dominio.
